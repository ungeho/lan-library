# LAN Library

LAN 内専用のローカルライブラリ/漫画ビュワー。  
ZIP対応・ドラッグ&ドロップ追加・読了率管理などを備え、環境変数によりNASや外部ストレージにも対応。

## 特徴

- 完全ローカル（外部通信なし）を想定
- LAN 内の複数端末からアクセス可能
- サーバー1つで家中の端末から閲覧

## セットアップ

```bash
npm install
npm run dev
```

- フロントエンド: http://localhost:5173
- API サーバー: http://localhost:3001

LAN 内の他端末: `http://<ホストIP>:5173`

## ライブラリ構成

`library/` フォルダに作品フォルダまたは ZIP ファイルを配置:

```
library/
├── 作品A/          ← フォルダ形式
│   ├── 001.jpg
│   ├── 002.png
│   └── ...
├── 作品B.zip       ← ZIP 形式
├── 作品C/
│   └── ...
```

対応形式:
- **画像**: JPG, PNG, WebP, GIF, AVIF
- **アーカイブ**: ZIP (中に画像が含まれていれば自動認識)

フォルダや ZIP を追加するとアプリが自動検出します。  
タイトル・セクション・シリーズ・巻数・カテゴリはアプリ内の編集画面で設定できます。

## 機能

### 管理
- タイトル・セクション・カテゴリ・シリーズで検索/フィルタ
- ソート機能
- メタデータ編集 (タイトル、セクション、シリーズ、巻数、カテゴリ)
- 一括編集
- 評価機能
- 読了率の表示・管理

### 取り込み
- フォルダ / ZIP 両対応
- `library/` 配下の自動検出
- 環境変数 `LIBRARY_DIR` によりライブラリの場所を自由に変更可能
- ZIP のドラッグ&ドロップによる直接追加

### 閲覧
- 本棚表示 (セクション別タブ + シリーズ別グループ化)
- ページビュワー (見開き/単ページ、右開き/左開き)
- ダークモード
- 全画面モード
- レスポンシブ対応

## ビュワー操作

| キー | 操作 |
|------|------|
| ← → | ページ移動 (読み方向に連動) |
| Space | 次ページ |
| F | 見開き切替 |
| Esc | UI 表示/非表示 |

画面の左1/3 = 前、右1/3 = 次、中央 = UI切替

## 環境変数

| 変数 | 説明 | デフォルト |
|------|------|-----------|
| `PORT` | サーバーポート | 3001 |
| `LIBRARY_DIR` | ライブラリパス | `./library` |
